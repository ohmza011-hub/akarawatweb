🌤️ สภาพอากาศเชียงใหม่ — Open-Meteo API

Open-Meteo เป็น API พยากรณ์อากาศแบบฟรี ไม่ต้องสมัครหรือขอ API key ใช้ดึงข้อมูลอุณหภูมิและสภาพอากาศปัจจุบันของพิกัดที่กำหนด ในหน้านี้ตั้งพิกัดไว้ที่จังหวัดเชียงใหม่

💬 คำคมประจำวัน — DummyJSON Quotes API

DummyJSON เป็น API ฟรีสำหรับทดสอบ/สาธิตการดึงข้อมูล มีชุดข้อมูล "คำคม" (quote) ให้สุ่มดึงมาแสดงได้ทันที ไม่ต้องขอ API key และรองรับการเรียกจากเบราว์เซอร์โดยตรง (CORS) เหมาะสำหรับใส่ในเว็บส่วนตัวเพื่อสร้างแรงบันดาลใจให้ผู้เข้าชม

💡 คำแนะนำสุ่ม — Advice Slip API

Advice Slip API เป็น API ฟรีที่ส่งคืน "คำแนะนำ" สั้น ๆ แบบสุ่มหนึ่งประโยคทุกครั้งที่เรียกใช้ ไม่ต้องใช้ API key เช่นกัน เหมาะกับการทำวิดเจ็ตให้กำลังใจเล็ก ๆ บนเว็บ
